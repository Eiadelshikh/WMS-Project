 WMS Testing Project

This project contains manual and automated test cases for a Warehouse Management System (WMS).

 Scope :
- Receiving (Inbound)
- Picking & Shipping (Outbound)
- Inventory Management
- User Roles & Permissions

 Types of Testing :
- Manual Testing
- API Automation
- Data base Testing
- Load Test

Tools (Planned):
- Postman / REST Assured (API)
- Table plus for Data base validation 
- GitHub Issues for bug tracking
- Jmeter for load test
